## 💡 What is PitchForge?
PitchForge helps freelancers and sales professionals write
highly personalized cold emails and DMs that actually get
replies — powered by Claude AI.

Paste your service + your prospect's info and get 3 battle-tested
pitch variations instantly:
- **The Problem-First** — leads with their pain point
- **The Social Proof** — builds credibility fast
- **The Curiosity Hook** — makes them want to reply

## ✨ Features
- ⚡ AI-generated pitches in under 5 seconds
- ✉️ Cold Email + Cold DM modes
- 🎯 4 tone options: Professional, Casual, Bold, Friendly
- 🔒 Free tier (3 pitches) + Paid plans
- 📱 Fully responsive design

## 🛠 Tech Stack
- **Frontend:** React 18 + Vite
- **AI:** Claude API (claude-sonnet)
- **Hosting:** Vercel
- **Payments:** Stripe
